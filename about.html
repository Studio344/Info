Studio344 とは

Studio344は、テクノロジーとものづくりが好きな個人の趣味アカウントです。仕事とは別に、興味のある技術を自由に試したり、個人開発を楽しむための場として運営しています。「面白そう」と思ったらまず手を動かす。そんなスタンスで、Webアプリやツールなどを気ままに作っています。

Profile

某グローバルIT企業でエンジニアとして勤務。業務外の時間で、気になった技術（React, Next.js, WebGLなど）を触ったり、日常のちょっとした不便を解消するツールを作ったりしています。

Technical Skills

Languages
JavaScript / TypeScript, Python, Java, Go
Frontend
React, Next.js, Vue.js, TailwindCSS, GSAP, Three.js
Backend
Node.js, Express, Supabase
Tools
Docker, Git, Vercel, Cloudflare

Philosophy

"Just Build It."

完璧を目指すより、まず動くものを作ってみる。趣味だからこそ、失敗を気にせず新しいことに挑戦できると思っています。
