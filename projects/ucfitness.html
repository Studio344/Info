UCFitness

Fitbit APIを活用した、グループ対抗フィットネス・トラッキング・プラットフォーム

Visit Live Site →
ucfitness.studio344.net

Core Features

UCFitness は「究極のソーシャルフィットネスダッシュボード」として、以下の3つのコア機能を提供します。

リアルタイムランキング

順位をリアルタイムで追跡。日次、週次、月次で同僚や友人と歩数・活動量を比較できます。

グループバトル

グループに参加してチーム対抗戦。個人のスコアだけでなく、チーム全体の合計値で競い合いましょう。

バッジ獲得

マイルストーン到達や連続記録（ストリーク）でユニークなバッジをアンロック。成果を可視化します。

Tech Stack Detail

最新のモダンWeb技術スタックを採用し、高速かつスケーラブルなアプリケーションを構築しました。

Frontend Framework: Next.js (App Router) - サーバーコンポーネントによる高速な初期表示を実現。
UI Library: TailwindCSS & Shadcn/UI - 美しく一貫性のあるデザインシステムを構築。
Authentication: OAuth 2.0 (Fitbit Web API) - セキュアなトークン管理と自動リフレッシュ機能。
Database & Realtime: Supabase (PostgreSQL) - ランキングのリアルタイム更新（Live Subscription）に使用。
Deployment: Vercel + Cloudflare - エッジネットワークを活用したグローバル配信。

Architecture Highlights

Fitbit API Rate Limit 対策:

ユーザー数が増加してもAPI制限に抵触しないよう、独自のキャッシングレイヤーを実装しています。個別のユーザーリクエストごとにAPIを叩くのではなく、サーバーサイドワーカーが定期的にデータを収集・集計し、データベースを更新する「非同期集計アーキテクチャ」を採用しました。
