Bento Gridデザインを採用した、没入感のあるポートフォリオサイト

View Source Code →

Design Concept

「モダンで未来的、かつ情報の視認性を損なわない」をコンセプトに設計されました。近年トレンドのBento Grid（弁当箱のようなグリッドレイアウト）を採用し、異なるサイズのコンテンツを整然と配置しています。

Tech Stack

  • Core: HTML5, CSS3 (Variables, Grid, Flexbox), Vanilla JavaScript
  • Animation: GSAP (GreenSock Animation Platform) for smooth transitions
  • Effects: CSS Filter Effects (Blur, Noise), SVG Filters
  • Hosting: GitHub Pages

Key Features

1. パフォーマンス重視のVanilla実装

ReactやVueなどのフレームワークを使用せず、素のJavaScriptとCSSで実装することで、最軽量かつ高速な読み込みを実現しています。不要なバンドルサイズを極限まで削減しました。

2. ダイナミックな視覚効果

マウスの動きに追従する「スポットライト効果」や、背景で揺らぐ「オーロラ・オーブ」をCSS変数とJSの連携で実装。Canvasを使用せずDOM操作のみで実現することで、モバイルバッテリーへの負荷も考慮しています。

3. i18n（国際化）対応

軽量な翻訳ライブラリを活用し、日本語と英語の切り替えをスムーズに行える設計にしています。
